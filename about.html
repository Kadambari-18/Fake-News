About This Project

🚀 This AI-powered Fake News Detector helps identify whether a news article is real or fake using advanced Machine Learning techniques.

🧠 The system analyzes linguistic patterns, word usage, and contextual data to provide accurate predictions with confidence scores.

🎤 It supports multiple input methods including text input, URL-based detection, and voice recognition for a seamless user experience.

📊 The goal of this project is to combat misinformation and promote awareness by providing users with reliable news verification tools.

👩‍💻 Developed as part of an internship project, this application demonstrates real-world implementation of Artificial Intelligence in solving modern problems.
